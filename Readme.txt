# 💼 Personal Portfolio Website

This is my personal portfolio website built with **HTML**, **CSS**, and **JavaScript**.  
It showcases my skills, projects, resume, and contact details — all in one place.

## 📁 Project Structure

├── index.html # Homepage
├── about.html / resume.html (if any)
├── assets/
│ ├── css/ # Stylesheets
│ ├── js/ # JavaScript files
│ ├── img/ # Images
│ └── vendor/ # External libraries (Bootstrap, AOS, etc.)

## 🚀 Features

- Smooth scroll and animations (AOS library)
- Responsive design with Bootstrap
- Downloadable resume
- Contact form (can be connected to Formsubmit or PHP)
- Social media links

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- AOS (Animate On Scroll)
- Git & GitHub
